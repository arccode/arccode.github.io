  1. 在阿里云管理控制台启动购买的云主机
  2. 使用zoc6连接服务器

  3. 更新软件 yum -y update

  4. 安装开发工具 yum -y groupinstall "Development Tools"

  5. 下载Node源码

    • wget http://nodejs.org/dist/v0.10.32/node-v0.10.32.tar.gz
  6. 安装Node
    • 解压node-v0.10.32.tar.gz
    • cd node-v0.10.32
    • ./configure
    • make
    • make install
    • *注：编译后的二进制文件会放置于系统路径下/usr/local/bin/node

Mac下安装memcached

发表于 2014-08-10   |   分类于 NoSQL(缓存)   |  

安装libevent(memcached的依赖库)

  • 下载libevent源码
  • 解压tar -xzf libevent-2.0.21-stable.tar.gz
  • cd libevent-2.0.21-stable
  • sudo ./configure --prefix=/usr
  • sudo make
  • sudo make install

安装memcached

  • 下载memcached源码
  • 解压memcached-1.4.20
  • cd memcached-1.4.20
  • sudo ./configure --prefix=/usr
  • sudo make
  • sudo make install
